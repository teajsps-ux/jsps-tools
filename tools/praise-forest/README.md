# Praise Forest

讚美森林 / 讚美讓小樹茁壯工具整理區。

## 目前原則

- 這個工具統一放在 `G:\我的雲端硬碟\jsps-tools\tools\praise-forest\`。
- 舊資料夾 `G:\我的雲端硬碟\讚美森林` 先保留，不刪除。
- 先建立本機可檢查檔案，不直接修改 Google Apps Script。
- 最終 Google Sheet / Apps Script 檔案留在 Google Drive，不把試算表內容、名單、驗證碼或 Apps Script 原始碼推到 GitHub。
- GitHub 只放師生入口與家長 / 老師線上入口網址。
- `gallery.html` 是老師 / 家長欣賞頁，不放新增讚美工具。
- `index.html` 是師生入口，才放新增讚美卡和老師工具。
- 先備份、再整理、再小步修正。

## 來源版本

- 基礎版：`讚美讓小樹茁壯v6.7(夢想森林版)`
- 目前可用測試版：`讚美讓小樹茁壯 v6.8 測試版`

`v6.8 測試版` 是從 v6.7 複製出來，為避免改壞而建立的備份測試版，且目前小地方修改成功。接下來先以 v6.8 測試版作為整理來源。

目前 `Code.gs` 內的學生姓名與驗證碼是已畢業學生 / 範例資料，用於工具測試與示範。

## 檔案規劃
