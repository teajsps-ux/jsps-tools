# 修復說明

## ✅ 已修復

### 1. 鍵盤方向鍵切換
- 添加了鍵盤監聽器，現在可以按 ← → 或 → ← 鍵來切換頁面
- 右下方的按鈕仍然可以正常使用

### 2. 對照頁亮燈功能
- 重寫了 `highlightLine` 函數，確保原詩與白話對照時能正確亮燈
- `playOriginalOnly()` - 點擊「讀原詩」，只亮原詩（左邊）
- `playPlainOnly()` - 點擊「讀白話」，只亮白話（右邊）
- `playCompare()` - 點擊「原詩白話一起讀」，同時亮兩邊

### 3. 其他修復
- 刪除了第 416 行多餘的 `}`，修復了 JavaScript 語法錯誤
- 修正了 `showWord` 函數的語音邏輯

## 測試方法

1. **鍵盤測試**
   - 打開 `index.html`
   - 按 ← → 鍵，應該能循環切換頁面

2. **對照頁測試**
   - 進入「原詩與白話對照」頁面
   - 點擊「讀原詩」按鈕，應該只有原詩行亮燈（變黃色）
   - 點擊「讀白話」按鈕，應該只有白話行亮燈（變黃色）

3. **瀏覽器檢查**
   - 按 F12 開啟開發者工具
   - 切換到 Console 標籤
   - 應該看到：「鍵盤監聽器已添加！按 ← → 測試。」
